## 🔹 一、基础风控规则
### 1. 仓位控制
| 参数 | 默认值 | 配置项 | 说明 |
|------|--------|--------|------|
| 单票最大仓位 | 20% | `max_position_per_stock` | 单只股票持仓占总权益的比例上限，默认最多同时持有5只股票 |
| 总仓位上限 | 70% | `max_position` | 总持仓市值占总权益的比例上限，固定保留30%现金 |

### 2. 持仓期限控制
| 参数 | 默认值 | 配置项 | 说明 |
|------|--------|--------|------|
| 最长持仓天数 | 3天 | `max_hold_days` | 任何股票持仓超过3天强制平仓，不受调仓日限制 |
| 强制平仓优先级 | 最高 | - | 每日盘前先检查强制平仓，再处理其他逻辑 |

### 3. 动态止损止盈
| 情绪等级 | 止损系数 | 止盈系数 | 说明 |
|----------|----------|----------|------|
| 极致冰点 | 0.8 | 0.9 | 止损收窄20%，止盈降低10% |
| 冰点 | 0.9 | 0.95 | 止损收窄10%，止盈降低5% |
| 修复期 | 1.0 | 1.0 | 正常参数 |
| 发酵期 | 1.1 | 1.05 | 止损放宽10%，止盈提高5% |
| 高潮期 | 1.2 | 1.1 | 止损放宽20%，止盈提高10% |

**基础阈值（可配置）：**
- 基础止损：5%（`stop_loss_pct`）
- 基础止盈：10%（`take_profit_pct`）

## 🔹 二、强制空仓规则（一票否决）
触发以下任一条件，当日自动卖出所有可卖出持仓，不再开新仓：
1. **大盘暴跌**：上证指数/创业板指单日跌幅≥3%
2. **恐慌情绪**：全市场跌停家数≥50只
3. **无赚钱效应**：日涨停家数<10只

## 🔹 三、情绪周期算法（专业实盘版）
### 1. 评分体系（0-100分）
| 指标类别 | 权重 | 包含指标 |
|----------|------|----------|
| 涨跌停类 | 45% | 涨停家数(25%)、跌停家数(10%)、炸板率(7%)、连板高度(3%) |
| 市场广度类 | 25% | 涨跌家数差(15%)、涨跌幅中位数(10%) |
| 资金流向类 | 25% | 北向资金净流入(12%)、龙虎榜净买入总额(8%)、全市场成交额(5%) |
| 波动率类 | 5% | 预留扩展VIX/溢价率等指标 |

### 2. 5档情绪等级
| 分数区间 | 情绪等级 | 仓位上限 | 允许策略 |
|----------|----------|----------|----------|
| 0~20 | 极致冰点 | 10% | 仅龙头低吸 |
| 20~40 | 冰点 | 30% | 龙头低吸、半路追涨 |
| 40~60 | 修复期 | 70% | 龙头低吸、半路追涨、首板打板 |
| 60~80 | 发酵期 | 90% | 增加涨停开板策略 |
| 80~100 | 高潮期 | 100% | 全策略开放（增加跌停翘板） |

## 🔹 四、9层筛选流程
| 层级 | 名称 | 过滤规则 | 过滤比例 |
|------|------|----------|----------|
| 1 | 强制空仓检查 | 触发条件直接空仓 | - |
| 2 | 特殊时期过滤 | 节假日/重大会议自动调整仓位 | 0~100% |
| 3 | 情绪周期适配 | 动态调整仓位上限和允许策略列表 | 0~100% |
| 4 | 盘前预选池 | 排除ST、次新股、流动性不足标的 | ~90% |
| 5 | 竞价阶段过滤 | 竞价涨幅0.5%~7%、未匹配量为正、竞价量合格 | ~80% |
| 6 | 量能/性质过滤 | 量能放大要求、排除庄股/僵尸股/出货标的 | ~50% |
| 7 | 盘中信号触发 | 匹配对应策略的入场条件 | ~30% |
| 8 | 时间窗口过滤 | 仅在高胜率时间段交易（早盘/尾盘） | ~20% |
| 9 | 综合校验 | 龙虎榜/北向/股性综合因子验证 | ~10% |

## 🔹 六、配置参数说明
### 回测配置示例
```python
config = {
    "universe": "all_a",
    "start_date": "20260105",
    "end_date": "20260320",
    "initial_cash": 1000000,
    "rebalance_freq": "daily",  # 日度调仓（超短策略）
    "top_n": 5,  # 每日最多选5只股票
    
    # 基础风控参数
    "max_position": 0.7,  # 总仓位70%
    "max_position_per_stock": 0.2,  # 单票20%
    "max_hold_days": 3,  # 最长持仓3天
    "stop_loss_pct": 0.05,  # 基础止损5%
    "take_profit_pct": 0.1,  # 基础止盈10%
    
    # 功能开关
    "enable_force_empty": True,  # 开启强制空仓
    "enable_sentiment_cycle": True,  # 开启情绪周期
    "enable_auction_filter": True,  # 开启竞价过滤
    
    # 流动性参数
    "liquidity_threshold": 5000000,  # 500万成交额门槛
    "volume_threshold": 1.5,  # 量能放大1.5倍
}
```
