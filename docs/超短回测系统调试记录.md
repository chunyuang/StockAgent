## 五、日志规范（实盘级审计标准）
所有日志必须包含具体数值，禁止省略号，格式统一：
```
[HH:MM:SS] 【阶段】具体内容: 数值/结果
```
必须包含的日志环节：
1. 任务启动：参数完整显示（时间、资金、策略、参数）
2. 初始化：管理器、数据源、交易日历
3. 逐日处理：
   - 当日日期、进度（第N天/共M天）
   - 股票池获取：原始数量、过滤后数量
   - 因子计算：完成状态、记录数
   - 策略筛选：各策略筛选前后数量、原因
   - 选股结果：选中股票列表、原因
   - 调仓执行：买卖记录（股票、数量、价格、金额、原因）
   - 当日结算：收益率、市值、持仓
4. 结果汇总：绩效指标、交易统计、分析结论
