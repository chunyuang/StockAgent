## 1. 开发环境

- **兼容**：Debian 12+、CentOS 8+、macOS 13+

- **要求**：Python 3.10+
- **推荐**：Python 3.11 / 3.12

### 1.3 开发工具

- **IDE 推荐**：VSCode + Python 插件
- **代码格式化**：black 或 autopep8
- **类型检查**：mypy (可选)

## 3. 运行环境

### 3.1 必需服务

| 服务 | 版本要求 | 说明 | 默认端口 | 是否必须 |
|------|---------|------|---------|---------|
| **Redis** | 5.0+ | 缓存、去重、状态存储 | 6379 | ✅ 必须 |
| **MongoDB** | 4.0+ | 数据持久化存储 | 27017 | ✅ 必须 |

### 3.2 服务配置说明

#### Redis

- **配置**：默认无密码，如有密码需要在 `.env` 中配置 `REDIS_PASSWORD`
- **持久化**：开启 RDB 持久化即可，不需要太频繁
- **内存**：因为主要是缓存，128MB 足够

#### MongoDB

- **数据库**：默认 `stock_agent`，可通过 `MONGO_DB` 环境变量修改
- **认证**：如果开启认证，配置 `MONGO_USER` 和 `MONGO_PASSWORD`
- **索引**：程序启动时会自动创建必需索引

### 3.3 网络要求

- 能够访问互联网，用于获取数据
- 需要能够访问以下域名：
  - `api.tusharepro.com` - TuShare API
  - `1.eastmoney.com` - 东方财富数据 (AKShare)
  - `baostock.org` - Baostock 数据
  - `open.feishu.cn` - 飞书 API (如果使用多维表格)

## 4. 外部接口依赖

### 4.1 TuShare

**官网**：https://tushare.pro/

- **需要**：注册账号获取 Token
- **权限**：获取涨跌停需要 `stk_limit` 接口权限，需要 2000 积分
- **免费积分**：新用户注册赠送 100 积分，需要每日签到攒积分
- **备选方案**：即使没有权限，三级 fallback 机制也能使用 AKShare 正常工作

**配置**：
```bash
TUSHARE_TOKEN=your_token_here
```

### 4.2 AKShare

**官网**：https://akshare.akfamily.xyz/

- **完全免费**：不需要注册，不需要 Token
- **限制**：涨跌停接口只能获取最近约 30 天数据
- **使用方式**：直接 pip 安装即可使用

### 4.3 Baostock

**官网**：http://baostock.com/

- **完全免费**：不需要注册
- **优点**：提供完整A股历史日线数据，从上市至今
- **使用场景**：回测需要完整历史数据时使用

### 4.4 飞书开放平台

**需要配置**（可选，如果要使用多维表格自动写入）：

1. 创建飞书自建应用
2. 授予权限：`bitable:app:read`、`bitable:app:write`
3. 获取 `APP_ID` 和 `APP_SECRET`
4. 配置到 `.env`：
   ```bash
   FEISHU_APP_ID=cli_xxx
   FEISHU_APP_SECRET=xxx
   FEISHU_BITABLE_APP_TOKEN=xxx
   FEISHU_BITABLE_TABLE_ID=xxx
   ```

**如果不配置**：只影响多维表格自动写入，Webhook 推送不受影响

### 4.5 飞书群聊 Webhook

**配置**（可选，如果要推送消息到群聊）：

```bash
NOTIFY_WECOM_WEBHOOK=https://open.feishu.cn/open-apis/bot/v2/hook/xxx
```

**说明**：企业微信和飞书 Webhook 格式兼容，所以变量名保留 `WECOM`

## 5. 数据库设计

### 5.1 MongoDB 集合结构

#### `stock_basic` - 股票基础信息

| 字段 | 类型 | 说明 |
|------|------|------|
| `ts_code` | str | 股票代码 (索引) |
| `symbol` | str | 简码 |
| `name` | str | 名称 |
| `area` | str | 地区 |
| `industry` | str | 行业 |
| `list_date` | int | 上市日期 |
| `delist_date` | int | 退市日期 |
| `is_delisting` | bool | 是否退市 |

#### `daily_kline` - 日线数据缓存

| 字段 | 类型 | 说明 |
|------|------|------|
| `ts_code` + `trade_date` | 复合索引 | |
| `open` | float | 开盘 |
| `high` | float | 最高 |
| `low` | float | 最低 |
| `close` | float | 收盘 |
| `pre_close` | float | 昨收 |
| `vol` | int | 成交量 |
| `amount` | float | 成交额 |
| `pct_chg` | float | 涨跌幅 |

#### `strategy_subscriptions` - 策略订阅

| 字段 | 类型 | 说明 |
|------|------|------|
| `_id` | ObjectId | 主键 |
| `strategy_name` | str | 策略名称 |
| `watch_list` | array | 观察列表 |
| `is_all_market` | bool | 是否全市场 |
| `parameters` | object | 策略参数 |
| `enabled` | bool | 是否启用 |

#### `alerts` - 预警记录

| 字段 | 类型 | 说明 |
|------|------|------|
| `_id` | ObjectId | 主键 |
| `subscription_id` | str | 订阅ID |
| `strategy_name` | str | 策略名称 |
| `strategy_subtype` | str | 策略子类型 |
| `ts_code` | str | 股票代码 |
| `trigger_price` | float | 触发价格 |
| `current_price` | float | 当前价格 |
| `trigger_reason` | str | 触发原因 |
| `position_ratio` | float | 建议仓位 |
| `created_at` | datetime | 创建时间 |

## 7. 配置文件说明

### 7.1 `.env` 文件位置

`/path/to/StockAgent/AgentServer/.env`

### 7.2 完整配置模板

```env
# Redis 配置
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=

# MongoDB 配置
MONGO_HOST=localhost
MONGO_PORT=27017
MONGO_USER=
MONGO_PASSWORD=
MONGO_DB=stock_agent

# TuShare 配置
TUSHARE_TOKEN=your_tushare_token_here

# 飞书配置 - 多维表格自动写入
FEISHU_APP_ID=your_app_id_here
FEISHU_APP_SECRET=your_app_secret_here
FEISHU_BITABLE_APP_TOKEN=HczLbin53a8zo7sjLEpcZ58QnEd
FEISHU_BITABLE_TABLE_ID=tblxLHQxpB424qP3

# 通知 Webhook (飞书/企业微信)
NOTIFY_WECOM_WEBHOOK=https://open.feishu.cn/open-apis/bot/v2/hook/your_webhook_key
```

### 7.3 必需配置项

至少需要配置：

```
REDIS_HOST, REDIS_PORT
MONGO_HOST, MONGO_PORT, MONGO_DB
```

其他配置项可以留空，对应功能会自动禁用（但不影响核心选股功能）

## 10. 常见环境问题

| 问题 | 解决方案 |
|------|---------|
| `ImportError: No module named 'baostock'` | `pip install baostock --break-system-packages` |
| `ImportError: No module named 'akshare'` | `pip install akshare --break-system-packages` |
| `Redis connection refused` | 启动 Redis：`redis-server --daemonize yes` |
| `MongoDB connection failed` | 启动 MongoDB：`systemctl start mongod` |
| `Tushare 积分不足` | 不影响，fallback 到 AKShare 正常工作 |
| `AKShare 返回空涨跌停` | 超过 30 天限制，正常，fallback 到昨日 |
| `飞书多维表格写入失败` | 检查 `FEISHU_APP_ID` 和 `FEISHU_APP_SECRET`，确认权限 |
