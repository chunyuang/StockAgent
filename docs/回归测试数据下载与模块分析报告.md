## 三、各个策略因子参数说明

| 因子 | 计算方式 | 参数 | 方向 |
|------|-------------|------|------|
| `limit_up_yesterday` | 昨日 pct_chg ≥ 9.5% → 1 = 是，0 = 否 | 昨日涨停 | 越大越好 |
| `limit_down_yesterday` | 昨日 pct_chg ≤ -9.5% → 1 = 是，0 = 否 | 昨日跌停 | 越大越好 |
| `open_below_limit` | 今日开盘价 < 昨日收盘价 * 1.095 → 1 = 是，0 = 否 | 开盘在涨停价以下 | 越大越好 |
| `open_above_limit` | 今日开盘价 > 昨日收盘价 * 0.905 → 1 = 是，0 = 否 | 开盘在跌停价以上 | 越大越好 |
| `first_limit_up` | 今日首次涨停 → 历史没有过涨停 → 1 = 是，0 = 否 | 首次涨停 | 越大越好 |
| `volume_increase` | 今日成交量 / 5 日平均成交量 → 越大越好 | 成交量放大 | 越大越好 |
| `price_near_ma5` | 偏离 MA5 越小得分越高 → 收盘价在 MA5 附近 | 价格接近 MA5 | 越大越好 |
| `market_leader` | 昨日涨停 + 成交量放大 → 1 = 是，0 = 否 | 市场龙头 | 越大越好 |
| `pullback_ma5` | 龙头前期涨停 → 今日回调到 MA5 附近 → 1 = 是，0 = 否 | 龙头回调 MA5 | 越大越好 |

### 核心问题分析

为什么会出现连续持有全部亏损？

**根本原因：**

我们在 **2026-01-05** 第一天选股 → 选出 5 只 → 买入之后 → 接下来每天：

- 每天都会重新选股 → 选出新的 5 只 → 卖出昨天的 → 买入今天的
- 所以每天都会完成 5 笔卖出 → 5 笔买入 → 每天都会产生 10 笔交易 → 为什么最终 total_trades = 0 ？

→ 因为回测最后一天 `2026-03-20` 你买入之后 → 没有下一个交易日卖出 → 所以这 5 笔买入没有完成 → 不统计 → 所以前面 47 天 → 每天都应该完成 10 笔交易 → 总交易应该 47 * 10 = 470 笔 → 为什么日志显示 total_trades = 0 ？
