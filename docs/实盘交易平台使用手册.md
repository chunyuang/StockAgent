# 🚀 StockAgent 实盘交易平台使用手册

## 📋 平台概述
这是一个全自动实盘量化交易平台，集成了策略信号生成、自动模拟交易、绩效分析、风控告警全链路功能，每日盘后自动运行，无需人工干预。

---

## ✅ 已上线功能
### 1. 📊 每日实盘信号生成
- **自动运行时间**：每个交易日15:30收盘后
- **包含内容**：
  - 市场情绪评分、等级、仓位建议
  - 当日允许交易的策略列表
  - 符合条件的选股标的列表
  - 完整交易计划（空仓/建仓/平仓建议）
- **推送渠道**：飞书群自动推送

### 2. 💼 自动模拟交易
- 实盘信号生成后自动执行交易
- 完全仿真A股交易规则：T+1、涨跌停限制、佣金印花税、滑点
- 自动仓位管理、买卖执行、持仓跟踪
- 支持多策略多账户同时运行

### 3. 📈 每日绩效报告
- **自动推送时间**：每个交易日15:40左右
- **报告内容**：
  - 账户概览：初始资金、当前余额、总收益、最大回撤
  - 当前持仓明细：股票代码、数量、成本、收益
  - 绩效指标：胜率、盈亏比、夏普比率、交易统计
- **推送渠道**：飞书群自动推送

### 4. 🔍 自动风控告警
- **监控维度**：
  - 账户最大回撤告警（默认阈值10%）
  - 单只股票亏损告警（默认阈值8%）
  - 仓位超限告警（默认阈值90%）
  - 单日亏损告警（默认阈值5%）
  - 市场异常波动告警
- **告警级别**：信息/警告/危险/严重，对应不同推送方式

## ⚙️ 自定义配置
### 修改风控阈值
编辑 `/root/.openclaw/workspace/StockAgent/real_trading/risk_alert.py` 中的 `default_config` 配置项：
```python
self.default_config = {
    "max_drawdown_alert": 0.1,  # 最大回撤超过10%告警
    "single_stock_loss_alert": 0.08,  # 单只股票亏损超过8%告警
    "position_limit_alert": 0.9,  # 仓位超过90%告警
    "daily_loss_alert": 0.05,  # 单日亏损超过5%告警
}
```

### 添加新的推送渠道
编辑 `/root/.openclaw/workspace/StockAgent/real_trading/signal_pusher.py` 配置对应的webhook地址。

### 创建新的模拟账户
```python
from real_trading.paper_trading import PaperTradingEngine
engine = PaperTradingEngine()
acc_id = engine.create_account('新账户名称', initial_balance=1000000)
```

## 📞 问题排查
### 定时任务未运行
1. 检查cron状态：`systemctl status cron`
2. 查看运行日志：`tail -f /root/.openclaw/workspace/StockAgent/logs/daily_signal.log`
3. 手动运行测试：`cd /root/.openclaw/workspace/StockAgent && ./scripts/daily_signal_job.sh`

### 推送失败
1. 检查网络连接
2. 确认飞书webhook配置正确
3. 手动运行脚本看报错信息

### 信号异常
1. 检查MongoDB服务是否正常运行
2. 确认日线数据是否更新到最新交易日
3. 查看回测引擎日志定位问题
