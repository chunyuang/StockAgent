## 一、删除的目录（整个目录移除）

### 1. `scripts/` (根目录，143个文件)
- **内容**: 3月份旧调试脚本（debug_*/check_*/fix_*/download_*/run_*/test_*等）
- **删除理由**: 全部是一次性调试脚本，功能已被AgentServer/scripts/替代
- **是否有用**: 极低。所有sys.path.insert指向已删除的workspace路径，无法直接运行
- **风险**: 如果需要查看3月份调试思路，可从git历史找回

### 2. `download_scripts/` (10个文件)
- **内容**: tushare下载脚本（download_daily_basic_tushare/download_lhb_tushare等）
- **删除理由**: tushare token已失效，所有脚本不可用
- **替代**: AgentServer/scripts/eastmoney_*.py

### 3. `local-databases/` (3.6M, 39个文件)
- **内容**: 旧版本地数据库管理（bin脚本+backtests/factors/raw元数据）
- **删除理由**: MongoDB已有所有数据，运行时代码不引用此目录
- **风险**: low。bin/new-db.py有硬编码路径引用workspace/local-databases（已不存在的），本身就不能用

### 4. `src/` (根目录，1个文件)
- **内容**: tools/local_data_service.py（BaseFetcher子类）
- **删除理由**: 无人import此文件。BaseFetcher已内联到AgentServer/src/tools/local_data_service.py
- **风险**: 无

### 5. `DOC/` (6个文件) → 合入 `docs/`

- backtest-log-analysis-20260429.md

### 6. `data/` (7个json)

### 7. `output/` (1个文件)
- **内容**: arkclaw任务产物task-15_indent_bug_review_report.md
- **删除理由**: 临时输出

### 8. `results/` (空目录)
