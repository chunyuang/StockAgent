# StockAgent 实时选股服务 - 项目介绍

## 项目概述

StockAgent 是一个基于 Python 的 **A股超短线游资策略实时选股服务**，核心目标是通过量化选股，结合游资情绪博弈，在 1-1.5 年内将 5000 元本金增长到 10 万元，验证超短线量化体系的盈利能力。

### 核心目标

- **体系搭建**：构建稳定可扩展的超短线量化选股体系
- **策略验证**：验证五大类游资策略的实际盈利能力
- **实盘执行**：通过信号推送辅助实盘交易
- **目标收益**：5000元 → 10万元 (20倍) 在 1-1.5 年内完成

### 核心特点

- **多数据源 fallback**：TuShare → AKShare → 新浪爬虫，三级 fallback 保证数据可用性
- **满足昨日涨停选股**：今日无数据时自动 fallback 到 AKShare 昨日涨停接口，满足"昨日涨停今日开盘选股"需求
- **完整飞书集成**：信号推送飞书群 + 自动写入飞书多维表格
- **模块化架构**：各数据源、策略相互独立，易于扩展
- **异步架构**：基于 asyncio 的异步架构，性能优秀

```

│  │  涨跌停数据  │ →  │   数据过滤   │ →  │   实时行情获取   │  │
│  │  (三级fallback)│    │            │    │  (新浪/TuShare)  │  │

│                                                                  │
│  ┌────────────────────────────────────────────────────────────┐  │

│                                                                  │
│  ┌──────────────┐    ┌──────────────┐    ┌──────────────────┐  │
│  │  策略评估    │ →  │  触发信号    │ →  │   推送飞书       │  │
│  │  (五大策略)   │    │            │    │   Webhook + 多维表  │  │
```

### 架构分层

| 层级 | 模块 | 职责 |
|------|------|------|
| **数据层** | TushareManager/AKShareManager/BaostockManager/SinaSpiderManager | 数据获取，标准化输出 |
| **管理层** | RedisManager/MongoManager/NotificationManager/FeishuBitableManager | 基础设施和外部服务 |
| **策略层** | BaseStrategy + 各策略实现 | 条件判断，信号生成 |
| **节点层** | ListenerNode | 轮询，调度，执行 |
| **输出层** | 飞书Webhook / 飞书多维表格 | 信号推送和记录 |

## 已实现策略

| 策略 | 代码文件 | 策略类型 | 说明 |
|------|---------|---------|------|
| 涨停开板/跌停翘板 | `limit_open.py` | 开盘选股 | 昨日涨停，今日开盘买入，涨停开板介入 |
| 5日线低吸/龙头低吸 | `ma5_buy.py` | 均线低吸 | 股价回踩5日线，偏差在一定范围内低吸 |
| 涨跌幅阈值/半路追涨 | `price_change.py` | 半路追涨 | 开盘涨跌幅达到阈值，追涨介入 |
| 龙头战法 | `leading_dragon.py` | 龙头股 | 市场龙头股低吸 |
| 首板打板 | `first_board.py` | 首板 | 首次涨停个股打板 |

## 数据源支持

| 数据类型 | 优先级 1 | 优先级 2 | 优先级 3 | 兜底 |
|---------|---------|---------|---------|------|
| **每日涨跌停列表** | TuShare `stk_limit` | AKShare 今日 | AKShare 昨日 | 东方财富爬虫 |
| **日线行情** | TuShare `daily` | AKShare | **Baostock** (新增免费完整历史) | - |
| **实时行情** | TuShare `quotes` | 新浪爬虫 | - | - |
| **龙虎榜数据** | - | AKShare 龙虎榜 | - | - |
| **股票基础信息** | TuShare `stock_basic` | AKShare | Baostock | - |

---

## 输出渠道

1. **飞书群聊 Webhook**：信号推送到指定飞书群
2. **飞书多维表格**：信号自动写入实时监控表（需要配置 AppID/AppSecret）
3. **MongoDB**：信号记录存储
