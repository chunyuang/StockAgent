# 当前安装的 Skill 功能概览

## 一、OpenClaw-Lark 扩展包 (飞书相关)

### 1. feishu-bitable
**功能**: 飞书多维表格管理
- 创建和管理多维表格应用
- 增删改查数据表、字段、记录
- 批量导入/导出数据
- 支持27种字段类型

### 2. feishu-calendar
**功能**: 飞书日历与日程管理
- 查询日历列表和主日历信息
- 创建、查询、更新、删除日程
- 管理参会人、回复日程邀请
- 查询忙闲状态

### 3. feishu-channel-rules
**功能**: 飞书频道输出规则
- 在飞书对话环境中自动应用规则
- 格式化消息输出
- 确保消息符合飞书格式要求

### 4. feishu-create-doc
**功能**: 创建飞书云文档
- 从Markdown内容创建文档
- 指定创建位置（文件夹/知识库）
- 支持异步任务查询

### 5. feishu-fetch-doc
**功能**: 获取飞书文档内容
- 提取文档的Markdown内容
- 处理文档中的图片、文件
- 支持大文档分页获取

### 6. feishu-im-read
**功能**: 飞书消息读取
- 获取群聊/单聊历史消息
- 搜索跨会话消息
- 下载图片/文件资源
- 读取话题回复

### 7. feishu-task
**功能**: 飞书任务管理
- 创建、查询、更新任务
- 管理任务清单
- 添加任务评论和子任务

### 8. feishu-troubleshoot
**功能**: 飞书插件问题诊断
- 常见问题FAQ查询
- 深度诊断配置问题
- 生成详细诊断报告

### 9. feishu-update-doc
**功能**: 更新飞书文档
- 支持7种更新模式：覆盖、追加、定位替换、全文替换、前后插入、删除范围
- 支持异步任务查询

## 二、Wecom-OpenClaw-Plugin 扩展包 (企业微信相关)

### 10. wecom-contact-lookup
**功能**: 通讯录成员查询
- 按姓名/别名搜索匹配员工
- 获取userid、姓名、别名信息
- 仅返回当前用户有权限查看的成员

### 11. wecom-doc-manager
**功能**: 企业微信文档管理
- 文档内容的读取和编辑
- 支持智能表格和文档
- 通过docid或URL定位文档

### 12. wecom-edit-todo
**功能**: 企业微信待办事项编辑
- 创建、更新、删除待办
- 变更用户处理状态
- 分派待办任务给其他用户

### 13. wecom-get-todo-detail
**功能**: 待办详情批量查询
- 根据待办ID列表获取完整信息
- 获取待办内容和分派信息
- 通常配合待办列表查询使用

### 14. wecom-get-todo-list
**功能**: 待办列表查询
- 按创建时间/提醒时间过滤
- 支持分页获取
- 仅返回待办概要信息

### 15. wecom-meeting-create
**功能**: 企业微信会议创建
- 预约会议
- 设置会议时间、参与者
- 生成会议链接

### 16. wecom-meeting-manage
**功能**: 企业微信会议管理
- 取消会议
- 更新会议参与人员
- 移除会议成员

### 17. wecom-meeting-query
**功能**: 企业微信会议查询
- 会议列表查询
- 会议详情获取
- 按关键词查找会议

### 18. wecom-msg
**功能**: 企业微信消息管理
- 会话列表查询
- 消息记录拉取
- 发送文本消息

### 19. wecom-preflight
**功能**: 企业微信前置条件检查
- 检查MCP工具权限配置
- 自动修复profile和白名单
- 需要时提示重启Gateway

### 20. wecom-schedule
**功能**: 企业微信日程管理
- 查询时间范围内的日程
- 创建新日程
- 修改/取消日程
- 添加/移除参与人

### 21. wecom-smartsheet-data
**功能**: 企业微信智能表格数据管理
- 查询子表全部记录
- 添加、更新、删除记录
- 通过docid或URL定位表格

### 22. wecom-smartsheet-schema
**功能**: 企业微信智能表格结构管理
- 查询子表列表
- 添加、更新、删除子表
- 管理字段/列定义

## 三、Workspace 基础 Skill

### 23. find-skills
**功能**: 技能发现与安装帮助
- 帮助用户搜索和发现可用skill
- 提供skill安装指导
- 展示常用skill分类

### 24. browser-use
**功能**: 浏览器自动化
- 网页浏览和截图
- 表单填写和数据提取
- 自动化操作流程

### 25. github
**功能**: GitHub操作
- issue和PR管理
- CI/CD运行控制
- 高级API查询

### 26. image-generate
**功能**: 图片生成
- 根据文本描述生成图片
- 支持多种风格和参数
- 直接发送到聊天会话

### 27. notion
**功能**: Notion管理
- 页面和数据库创建
- 块内容管理
- 文档结构和内容更新

### 28. self-improving-agent
**功能**: 自我改进记录
- 捕获学习和错误
- 记录用户纠正
- 持续改进建议

### 29. slack
**功能**: Slack操作
- 消息发送和回复
- 反应和交互操作
- 频道管理

### 30. smart-ocr
**功能**: 智能OCR文字识别
- 支持100+语言
- 图片和扫描文档识别
- 多语言文本提取

### 31. veadk-skills
**功能**: VeADK技能集合
- 完成VeADK相关功能
- Agent创建和转换
- 系统集成操作

### 32. veadk-go-skills
**功能**: VeADK-Go技能集合
- 根据需求生成Agent
- 将Enio Agent转换为VeADK-Go Agent
- 高级功能开发

### 33. video-generate
**功能**: 视频生成
- 根据文本描述生成视频
- 支持首帧图片设定
- 返回有效期为24小时的视频URL

### 34. web_search
**功能**: 隐私友好的元搜索
- 使用本地SearXNG实例
- 搜索网页、图片、新闻等
- 无外部API依赖

### 35. workspace-netdrive
**功能**: 工作区网盘管理
- 检测和管理挂载的网盘
- TOS存储桶挂载管理
- 持久化存储管理

## 四、其他核心扩展

### 36. memory-lancedb-ultra
**功能**: 超高性能向量记忆库
- 长期记忆存储和管理
- 高性能向量搜索
- 大容量记忆库

### 37. dingtalk-connector
**功能**: 钉钉连接器
- 钉钉消息对接
- 组织通讯录同步
- 工作流自动化

## 总结

**总计安装：37 个 skill**
- **飞书相关**：9 个 skill，覆盖文档、日历、任务、表格等核心功能
- **企业微信相关**：13 个 skill，覆盖通讯录、文档、会议、日程等完整工作流程
- **基础功能**：13 个 skill，包括浏览器、GitHub、图片/视频生成等
- **其他扩展**：2 个 skill，提供记忆管理和钉钉连接

这些skill提供了完整的企业协作工具集，特别适合：
1. **企业内部协作**：飞书、企业微信集成
2. **开发工作流**：GitHub、文档管理、代码质量
3. **自动化处理**：浏览器操作、OCR识别、内容生成
4. **系统管理**：记忆管理、文件存储、工作流程优化
