# 策略参数说明

## 当前4策略配置（涨停开板默认关闭）

### 半路追涨 (halfway_chase)
- **选股逻辑**: 量比≥2.0 + 涨幅2-5% + 涨幅低位买入
- **止损**: 2%（严格止损，主力策略）
- **止盈**: 7%
- **持仓**: 最长2天
- **胜率**: 65-71%（历史回测）

### 首板打板 (first_limit_up)
- **选股逻辑**: 首次涨停 + 封单≥5000手 + 10:00前封板 + 流通市值50-500亿
- **止损**: 5%（买入价≈涨停价，2%太紧必触发）
- **止盈**: 8.5%
- **持仓**: 最长2天
- **胜率**: 47-50%（5%止损后从亏转盈）

### 龙头低吸 (leader_buy_dip)
- **选股逻辑**: 近5日≥2连板 + 回调8-35% + 回调1-7天 + 量比≤1.5
- **止损**: 5%（回调买入需空间）
- **止盈**: 6%
- **持仓**: 最长4天
- **胜率**: 极低频但盈利

### 跌停翘板 (limit_down_qiao)
- **选股逻辑**: 2连跌停后翘板 + 换手率≥10% + 翘板金额≥1000万
- **止损**: 7%（极端波动需大空间）
- **止盈**: 7%
- **持仓**: 最长3天
- **胜率**: 极低频

### 涨停开板 (limit_up_open) - 默认关闭
- **原因**: 胜率<30%，5%止损后仍大幅亏损
- **选股逻辑**: ≥2连板后开板 + 开板5分钟内 + 封单≥3000手
- **如需启用**: 在回测页面手动勾选

## 策略级止损原理

| 策略 | 买入价位置 | 合理止损 | 原因 |
|------|-----------|---------|------|
| 半路追涨 | 涨幅2-5%处 | 2% | 价格已偏高但未极端 |
| 首板打板 | 接近涨停价 | 5% | 涨停价买入，日内波动5%+常见 |
| 涨停开板 | 涨停价附近 | 5% | 同上 |
| 龙头低吸 | 回调低点 | 5% | 回调买入需等待企稳 |
| 跌停翘板 | 跌停价附近 | 7% | 极端波动，需更大空间 |
