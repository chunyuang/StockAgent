## 四、常见问题排查
### 1. 回测报错排查流程
1. 查看Web节点日志：`tail -f /tmp/web_node.log`
2. 查看回测任务独立日志：`/root/.openclaw/workspace/StockAgent/logs/backtest/{task_id}.log`
3. 检查MongoDB数据是否完整：确认对应日期的K线和因子字段存在
4. 检查服务是否正常运行：`ps aux | grep AgentServer`
### 2. 服务重启命令
```bash
# 停止所有服务
ps aux | grep -E "AgentServer|backtest" | grep -v grep | awk '{print $2}' | xargs kill -9 2>/dev/null || true
lsof -i :8000 | grep LISTEN | awk '{print $2}' | xargs kill -9 2>/dev/null || true
# 启动Web节点
cd /root/.openclaw/workspace/StockAgent/AgentServer && nohup python main.py --node-type web > /tmp/web_node.log 2>&1 &
```
