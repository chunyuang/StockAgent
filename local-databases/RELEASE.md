# 项目6 - 本地数据库管理 - 发布版本

## 简介

这是一个通用的股票量化本地数据库管理框架，适用于：
- 量化回测，分级存储原始数据/因子计算/回测结果
- 一次下载永久复用，避免重复请求
- 自动一致性校验，避免低级错误
- 完整追溯链路，任何回测都能知道用了哪个版本的原始数据和因子

## 快速开始

1. 克隆/解压这个框架
2. 按照 `OPERATION.md` 操作手册操作
3. 添加你的原始数据 → 计算因子 → 回测 → 自动记录

## 特性

- ✅ 三级分级存储（原始/因子/回测）
- ✅ 一次下载永久复用，节省时间和避免限流
- ✅ 自动一致性校验（回测前必做）
- ✅ 自动增量更新每日数据
- ✅ 自动生成 dashboard 索引
- ✅ 自动记录操作变更日志
- ✅ 版本号管理
- ✅ 完整依赖追溯
- ✅ 自动备份/恢复 MongoDB
- ✅ 过期数据库提示归档
- ✅ 完整操作手册

## 文件清单

```
local-databases/
├── README.template.md     # README 模板（发布版用这个）
├── DATA_SOURCES.md         # 数据源规则/额度/擅长时间段

├── CHANGELOG.md            # 操作变更日志（空模板）
├── USAGE.md               # 使用说明 + 数据源优先级配置

├── RELEASE.md             # 本文件 - 发布说明
├── bin/
│   ├── check-db.py             # 一致性校验
│   ├── incremental-update.py  # 自动增量更新
│   ├── regenerate-index.py    # 自动生成 README
│   ├── log-change.py          # 自动记录变更日志
│   ├── list-unused.py         # 列出过期提示归档
│   └── backup-db.py          # 备份/恢复 MongoDB
├── raw/
│   ├── metadata/             # 原始数据库描述（空目录）
│   └── data/                 # 原始数据备份（空目录）
├── factors/
│   ├── metadata/             # 因子组合描述（空目录）
│   └── data/                 # 因子数据备份（空目录）
└── backtests/
    ├── metadata/             # 回测描述（空目录）
    └── results/              # 回测结果输出（空目录）
```
